# Phishing Detection Web Application 🔐

This project is a cybersecurity web application built using Python Flask.

The system detects phishing websites and suspicious YouTube videos.

## Features

- User Login & Registration
- Phishing Website Detection
- YouTube Video Phishing Analysis
- YouTube API Integration
- Video Thumbnail Display
- Channel Name & View Count
- Dashboard Interface

## Technologies Used

- Python
- Flask
- Machine Learning (Scikit-learn)
- HTML / CSS / Bootstrap
- YouTube Data API

## Installation

Install libraries:

```
pip install -r requirements.txt
```

## Run the Application

```
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
